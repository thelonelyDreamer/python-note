### 1. 继承基类实现多态
``` python
class Animal:
    def speak(self):
        pass


class Dog(Animal):
    def speak(self):
        print("汪汪汪")


class Cat(Animal):
    def speak(self):
        print("喵喵喵")


def speak(animal: Animal):
    animal.speak()


dog = Dog() 
cat = Cat()
speak(dog)
speak(cat)
```
